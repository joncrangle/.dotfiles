## Integrations
- **Tailwind CSS**: [`@kobalte/tailwindcss`](https://github.com/kobalte/kobalte/tree/main/packages/tailwindcss) - Adds `ui-*` modifiers.
- **Solid Transition Group**: Useful for animating mount/unmount of `Content`.

## Implementations
- **Opencode**: Uses Kobalte for web UI components (Toasts, Dialogs, Popovers).
- **Hope UI**: Styled component library built on top of Kobalte.
